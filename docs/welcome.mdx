# 招新简章

## 普通社员

无要求，直接来，有空记得参加社活。

## 核心社员

核心社员是学校信息学科竞赛的中坚力量，核心社员需要具备相应的学科素养，满足以下**任何一项**即可成为核心社员：

1. **通过入社面试**（最简单的方式）
2. 参加过 CSP/NOIp 系列比赛
3. 洛谷蓝名及以上或 Codeforces 蓝名及以上
4. 有相应竞赛经验（什么竞赛经验属于竞赛经验解释权归市二编程社）

以下情形**满足两项**即可成为核心社员：

1. 为我校【单片机/人工智能】拓展课学员
2. 参加过【[光启创新基地](https://qzx.xhedu.sh.cn/cms/app/info/cat/index.php/112)】活动
3. 有【[全国青少年科技创新大赛](https://castic.cyscc.org/)】计算机项目相关奖项

或者参加入社面试，面试**满足以下任意一项**即可成为核心社员：

1. **看得顺眼**（bushi
2. 随意挑选一种语言（可选范围见附录），能回答出我们关于该语言问的一些刁钻的问题（请相信提问者的能力，一定能问的你怀疑人生完全回答不上来）
3. 有能力通过洛谷`入门`难度随机抽题测试，仅需口述算法流程或直接写代码，具体允许使用语言见附录。随机抽题范围：[洛谷入门难度题库](https://www.luogu.com.cn/problem/list?difficulty=1)

#### 核心社员福利

1. **直接组队代表学校参加**重要学科竞赛（如 [YAIOC](https://ai.secsa.cn/)、交大荣昶杯等）

2. 如有需要可直接找社员组队参加学科竞赛。

3. 参加核心社员竞赛培训，参加完培训基本重大学科竞赛都能拿很好的成绩。
